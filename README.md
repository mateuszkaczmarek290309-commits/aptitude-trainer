# Aptitude Trainer

A numerical reasoning aptitude trainer built with Python and Flask. Designed to help users practise the type of maths questions found in graduate and apprenticeship aptitude tests.

## What it does

- Generates random numerical reasoning questions
- Gives you 3 attempts per question before moving on
- Tracks your score across 20 questions
- Shows your final score and percentage at the end

## Why I built it

I built this to help with my own aptitude test preparation for finance and tech apprenticeship applications.

## Technologies used

- Python 3
- Flask
- HTML/CSS/JavaScript

## How to run it

```bash
python3 -m venv venv
source venv/bin/activate
pip install flask
python3 app.py
```
